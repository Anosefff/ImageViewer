# ImageViewer

こちらは私がポートフォリオの1つとして作成した画像ビューアです。
 
# Features
 
MVVM(Model-View-ViewModel)パターンを採用しています。
画像加工処理はすべて添付ビヘイビアに定義してコードビハインドから切り離すことで、
再利用性の高いコードを実現しています。

画像ビューアとしての機能はシンプルになるように心がけました。
